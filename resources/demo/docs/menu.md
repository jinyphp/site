# 메뉴설정
지니PHP는 직관적으로 메뉴를 수정할 수 있는 위젯 기능을 제공합니다.

## 패키지설치
메뉴위젯을 사용하기 위해서는 `jiny/menu` 패키지를 설치해야 합니다. 컴포저를 통하여 설치가 가능합니다.

```php
composer require jiny/menu
```
