# card 레이아웃

card는 다양한 곳에서 유용하게 자주 사용되는 UI 입니다.

## 그리드 배치
카드 안에 있는 내용이 서로 달라도 동일한 폭과 높이를 유지하면서 격자로 배치를 할 수 있습니다.

```html
<x-col-6 class="mb-4">
                    <x-card class="h-100">
                        A
                    </x-card>
</x-col-6>
```

동일한 폭을 유지하기 위해서 카드의 높이를 100%로 설정합니다. 또한, 여러개의 카드를 컬럼그리드를 이영하여 배치를 할때에는 상위 부모 요소에 하단 마진을 부여해 주면 됩니다.
